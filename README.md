# Todo App Avancée avec React Hooks

## Description

Ce projet est une application React complète réalisée dans le cadre du TP :

**"Création d'une Application React Complète avec les Hooks"**

L’objectif du projet est de découvrir et pratiquer les hooks fondamentaux et avancés de React à travers une application interactive de gestion de tâches.

L’application inclut :
- gestion de tâches
- changement de thème
- navigation avec React Router
- optimisation des performances
- hooks personnalisés
- transitions React 18

---

# Technologies utilisées

- React.js
- React Router DOM
- JavaScript ES6+
- CSS3
- Hooks React

---

# Hooks utilisés

## Hooks fondamentaux
- useState
- useEffect
- useContext
- useReducer

## Hooks avancés
- useMemo
- useCallback
- useRef
- useLayoutEffect
- useImperativeHandle
- useDebugValue
- useTransition
- useDeferredValue

## Hooks personnalisés
- useOnlineStatus
- useLocalStorage
- autres hooks personnalisés

---

# Fonctionnalités principales

## Gestion des tâches
- Ajouter une tâche
- Modifier une tâche
- Supprimer une tâche
- Marquer une tâche comme terminée

## Gestion du thème
- Mode clair / sombre
- Utilisation de Context API

## Navigation
- React Router
- Plusieurs pages dynamiques

## Optimisation des performances
- Mémorisation avec useMemo
- Optimisation avec useCallback

## Interface réactive
- useTransition
- useDeferredValue

# Installation du projet

## 1. Cloner le projet

```bash
git clone <url-du-projet>
```

## 2. Accéder au dossier

```bash
cd todo-app-avancee
```

## 3. Installer les dépendances

```bash
npm install
```

## 4. Lancer l’application

```bash
npm start
```

L’application sera accessible sur :

```bash
http://localhost:3000
```
